# Smart Oil Maintenance System

AI-based Predictive Maintenance Dashboard for Oil & Gas Equipment.

## Features
- Failure Risk Prediction using Machine Learning
- Dynamic Risk Gauge Visualization
- Cost Estimation
- Root Cause Detection
- Feature Importance Analysis

## Tech Stack
- Python
- Random Forest
- Streamlit
- Plotly

## How to Run
pip install -r requirements.txt
streamlit run app.py
